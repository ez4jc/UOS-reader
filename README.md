# UOS-reader

一款专为统信UOS（ARM架构）开发的TXT文本阅读器，界面简洁，功能实用。

## 功能特性

- **TXT文件导入**：支持打开外部TXT文件，自动检测编码（UTF-8/GBK/GB2312等）
- **智能目录识别**：自动识别章节标题，支持"第X章"、"Chapter X"、"一、二、三"等多种格式
- **透明背景模式**：快捷键切换全透明背景，文字悬浮于桌面
- **全局热键**：支持后台隐藏，程序不在任务栏显示但仍可响应热键
- **系统托盘**：隐藏到托盘，最小化系统资源占用，点击托盘恢复
- **可配置快捷键**：所有快捷键均可通过设置界面自定义
- **无边框设计**：无标题栏窗口，任意位置可拖动移动
- **窗口位置记忆**：自动记住窗口位置、大小和上次阅读进度
- **滚轮翻页**：支持鼠标滚轮滚动阅读
- **滚轮调节字体**：支持Ctrl+滚轮调节字体大小
- **章节导航**：支持上一章/下一章快捷切换，目录列表快速跳转
- **外观自定义**：支持设置字体、颜色、透明度等
- **D-Bus接口**：暴露`com.zgyd.Reader`接口，支持外部程序调用

## 界面说明

- **普通模式**：显示菜单栏、工具栏、目录列表，可通过右上角按钮关闭/最小化
- **透明模式**：仅显示文字内容，窗口完全透明，任意位置拖动

## 默认快捷键

| 功能 | 快捷键 |
|------|--------|
| 透明背景切换 | Ctrl+Shift+T |
| 隐藏到托盘 | Ctrl+Shift+H |
| 打开文件 | Ctrl+O |
| 退出程序 | Ctrl+Q |
| 上一章 | ← |
| 下一章 | → |
| 增大字体 | Ctrl++ |
| 减小字体 | Ctrl+- |

## 安装

### 方式一：安装deb包

```bash
sudo dpkg -i reader_1.0.0_arm64.deb
```

### 方式二：从源码编译

```bash
# 安装Qt开发环境
sudo apt install qt5-default qtcreator

# 编译
qmake reader.pro
make

# 运行
./reader
```

## 配置

配置文件位于：`~/.config/reader.ini`

可配置的选项包括：
- 快捷键绑定（透明切换、隐藏托盘、文件操作、章节导航等）
- 字体大小、颜色、字体家族
- 背景透明度
- 默认编码
- 自动章节检测
- 上次阅读进度（文件、章节、滚动位置）

## 技术栈

- Qt 5.11
- C++11
- D-Bus
- X11（全局热键）
- 目标平台：统信UOS（ARM架构）
